Teconologia - realidade Aumentada

De uma forma simples, Realidade Aumentada é uma tecnologia de permi que o mundo virtual seja misturado com o real, possibilidade maior interação e abrindo uma nova dimessão na maniera como nós executamos tarefas, ou mesmo as que incubimos às maquinas. Assim, se voce pensa que o objetos pulando para fora de tela eram elementos de filme de ficção cientifico, está na hora de mudar seus conceitos. Aliás, o que acontece coma realidade aumentada é o contrario: você pulará para dentro do mundo virtual para interagir com objetos que só estão limitados à sua imaginação. - tecmundo.com.br/realidade-aumentada - acesso 09/2016

Acessibilidade

Acessibilidade na web significa que as pessoas com deficiencia podem usar a web. Mas especificamnete, a acessibilidade na web significa que as pessoas com deficiencia podem perceber, entender, navegar,interagir e contrbuir para a web. Ela também beneficia outras pessaos, incluindo pessoas idosas com capacidades em mundança devido ao envelhecimento - w3c.br - acesso 09/2016
